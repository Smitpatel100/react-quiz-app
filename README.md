#  React Multi-Quiz Application

A multi-topic quiz application built using React.  
Users can select different quiz categories and answer 10 questions per topic with real-time progress tracking and scoring.

---

##  Features

-  Multiple quiz topics (Java, SQL, React)
-  10 questions per topic (30 total questions)
-  One-question-at-a-time interface
-  Dynamic score calculation
-  Progress bar tracking
-  Result screen with percentage
-  Restart quiz functionality
-  Clean and responsive UI

---

##  Tech Stack

- React.js
- JavaScript (ES6)
- CSS
- Git & GitHub
